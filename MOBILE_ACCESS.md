# 移动端访问指南

## 当前服务器地址

服务器已启用 HTTPS，支持移动端访问：

- **本机访问**：https://localhost:3007
- **局域网访问**：https://10.1.48.28:3007

## 移动端访问步骤

1. 确保手机和电脑连接在同一个 Wi-Fi 网络
2. 在手机浏览器中输入：**https://10.1.48.28:3007**
3. 如果出现"不安全"警告：
   - **Chrome**: 点击"高级" -> "继续前往 10.1.48.28"
   - **Safari**: 点击"显示详细信息" -> "访问此网站"
   - **其他浏览器**: 类似操作，找到"继续访问"选项

## 常见问题

### "无法建立安全连接"
- 确保使用的是 `https://` 而不是 `http://`
- 检查手机和电脑是否在同一网络

### 白屏问题
如果页面加载后显示白屏：
1. 打开浏览器的开发者工具（如果有）
2. 查看 Console 是否有错误信息
3. 尝试刷新页面

### 证书警告
这是正常的，因为使用的是开发用的自签名证书。在生产环境中应该使用正式证书。

## 故障排除

如果无法访问：
1. 检查防火墙设置，确保端口 3007 开放
2. 尝试使用电脑的 IP 地址：`ipconfig` (Windows) 或 `ifconfig` (Mac/Linux)
3. 确保 Vite 服务器正在运行
